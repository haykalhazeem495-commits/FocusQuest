FocusQuest
Project Overview

FocusQuest is a productivity-based learning application designed to improve students’ study habits and concentration. The system combines productivity tracking with simple game elements to create a more engaging learning experience. The goal of FocusQuest is to help students stay focused during study sessions while improving learning efficiency through interactive features and progress tracking.

Sustainable Development Goal (SDG)

SDG 4: Quality Education

This project supports the United Nations Sustainable Development Goal 4 by promoting better learning access and encouraging students to develop effective study habits through the use of digital technology. Software solutions such as FocusQuest can help enhance learning engagement, support skill development, and make studying more structured and motivating for students.

Project Description

FocusQuest transforms traditional studying into a more engaging experience by integrating productivity tracking with gamification features. The system allows students to manage study sessions, track their progress, and stay motivated through a structured and interactive environment. The platform aims to encourage consistent study habits while improving students’ ability to focus and manage their learning time effectively.

Key Features

1) Study session tracking
2) Productivity monitoring
3) Gamification elements to motivate users
4) User-friendly interface for students
5) Progress visualization to track learning performance

Development Approach

The project was developed using the Agile Scrum methodology, which allows iterative development and continuous improvement throughout the project lifecycle.

The development process includes:

1) Requirement gathering
2) System design and modeling
3) Prototype development
4) Testing and evaluation
5) Final system documentation

Tools used in the project include:

1) Trello – Agile project management and sprint tracking
2) GitHub – Version control and project documentation
